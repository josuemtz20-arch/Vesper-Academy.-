How to run this 5–7 minute interview

  1. Warm up (30 sec): Greet the student and chat casually to relax them. Do not score this.
  2. Climb the ladder: Start ONE rung below their placement-test result. Ask the questions for that rung, then move up. You don't need to ask every question — ask enough to judge.
  3. Keep going up until the student clearly struggles to respond. Rate each rung: Coped well / Hesitant / Struggled.
  4. The tool finds the level & course: the true level is the highest rung the student handled comfortably. Each rung is a Vesper course, so the recommended course appears automatically. Trust what you hear over the test score.
  5. Score the rubric (overall impression) and write a short note. Then copy or print the report.
